- [CREATE] 새로 만들었을때
- [UPDATE] 만들어진 것에서 수정이나 변동사항이 생겼을때
- [ADD] 무언가가 추가 되었을때
- [DELETE] 만들어진 것을 삭제 했을때
- [REFACTORING] 코드를 리팩토링 했을때
- [FIX] 오류를 확인하고 고쳤을때
- [REPLACE] 파일을 옮겼을 때
